# Linux

## Dependencies

### Homebrew

```sh
brew install zsh curl git
```

### YUM

```sh
sudo yum check-update
sudo yum -y install zsh curl git
```

### APT

```sh
sudo apt update
sudo apt install -y zsh curl git
```

## Installation

```sh
curl -L http://install.ohmyz.sh | zsh
```

```sh
sudo chsh $USER -s /bin/zsh
```

```sh
sudo su - $USER
```
