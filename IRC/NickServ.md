# NickServ

## Nick

```sh
/nick [nickname]
```

## Register

```sh
/msg NickServ REGISTER [password] [email]
```

## Change Password

```sh
/msg NickServ SET PASSWORD [password]
```

## Change Email

```sh
/msg NickServ SET EMAIL [email]
```

## Status

```sh
/msg NickServ STATUS
```

## Info

```sh
/msg NickServ INFO
```

## Login

```sh
/msg NickServ IDENTIFY [nickname] [password]
```

## Logout

```sh
/msg NickServ LOGOUT
```

## Vacation

```sh
/msg NickServ VACATION
```
