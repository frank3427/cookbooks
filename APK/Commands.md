# Commands

## Install

```sh
sudo apk add [package]
```

### Without cache

```sh
sudo apk add --no-cache [package]
```

## Info

```sh
sudo apk info [package]
```

## Search

```sh
sudo apk search [package]
```

## Update

```sh
sudo apk update
```

## Upgrade

```sh
sudo apk upgrade
```

## Delete

```sh
sudo apk del [package]
```

## Clean

```sh
sudo apk clean all
```
