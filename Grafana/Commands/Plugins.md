# Plugins

## Install

```sh
grafana-cli plugins install [plugin-id]
```

## List

```sh
grafana-cli plugins ls
```

### Remote

```sh
grafana-cli plugins list-remote
```

## Update

```sh
grafana-cli plugins update [plugin-id]
```

### All

```sh
grafana-cli plugins update-all
```

## Remove

```sh
grafana-cli plugins remove [plugin-id]
```
