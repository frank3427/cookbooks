# Linux

## Installation

### YUM

```sh
sudo yum check-update
sudo yum -y install python34-pip
```

> Preferably to install through [SCL](https://www.softwarecollections.org)

### APT

```sh
sudo apt update
sudo apt install -y python3-pip
```

### APK

```sh
sudo apk update
sudo apk add py3-setuptools
```
