# Service

## Create

```sh
docker service create
```

## Inspect

```sh
docker service inspect [id]
```

## Logs

```sh
docker service logs [id]
```

## List

```sh
docker service ls
```

## Process

```sh
docker service ps [id]
```

## Remove

```sh
docker service rm [id]
```

## Rollback

```sh
docker service rollback [id]
```

## Scale

```sh
docker service scale
```

## Update

```sh
docker service update [id]
```
