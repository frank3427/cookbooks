# Commands

## List

```sh
pyenv install --list
```

## Install

```sh
pyenv install [version]
```

## Rehash

```sh
pyenv rehash
```

## Versions

```sh
pyenv versions
```

## Setting

```sh
pyenv local [version]
pyenv global [version]
```

## Uninstall

```sh
pyenv uninstall [version]
```

## Which

```sh
pyenv which python[version]
```

## Whence

```sh
pyenv whence 2to3
```
