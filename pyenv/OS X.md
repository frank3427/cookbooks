# OS X

## Dependencies

### Homebrew

```sh
brew install openssl readline zlib
```

> Run `macOS_SDK_headers_for_macOS_10.14.pkg` before continue.

## Installation

### Homebrew

```sh
brew install pyenv
```

## Environment

For Bash or Zsh, put something like this in your `$HOME/.bashrc` or `$HOME/.zshrc`:

```sh
# Pyenv
eval "$(pyenv init -)"
```

```sh
sudo su - $USER
```
