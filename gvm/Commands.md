# Commands

## List

```sh
gvm list
```

## Remote

```sh
gvm listall
```

## Install

### Source

```sh
gvm install [version]
```

### Binary

```sh
gvm install [version] -B
```

## Use

```sh
gvm use [version] [--default]
```

## Uninstall

```sh
gvm uninstall [version]
```

## Implode

```sh
gvm implode
```
