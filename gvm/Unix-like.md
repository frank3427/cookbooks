# Unix-like

## Installation

```sh
bash < <(curl -s -S -L https://raw.githubusercontent.com/moovweb/gvm/master/binscripts/gvm-installer)
```

## Environment

For Bash or Zsh, put something like this in your `$HOME/.bashrc` or `$HOME/.zshrc`:

```sh
# GVM
source $HOME/.gvm/scripts/gvm
```

```sh
sudo su - $USER
```
