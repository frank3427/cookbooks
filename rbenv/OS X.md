# OS X

## Dependencies

### Homebrew

```sh
brew install openssl readline
```

## Installation

### Homebrew

```sh
brew install rbenv
```

## Environment

For Bash or Zsh, put something like this in your `$HOME/.bashrc` or `$HOME/.zshrc`:

```sh
# rbenv
eval "$(rbenv init -)"
```

```sh
sudo su - $USER
```
