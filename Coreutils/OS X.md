# OS X

## Installation

### Homebrew

```sh
brew install coreutils
```

## Environment

### Homebrew

For Bash or Zsh, put something like this in your `$HOME/.bashrc` or `$HOME/.zshrc`:

```sh
# GNU Coreutils
export PATH="/usr/local/opt/coreutils/libexec/gnubin:$PATH"
```

```sh
sudo su - $USER
```
