# Commands

## Sign in

```sh
mas signin [email]
```

### Dialog

```sh
mas signin --dialog [email]
```

## List

```sh
mas list
```

## Search

```sh
mas search [name]
```

## Install

```sh
mas install [id]
```

## Lucky

```sh
mas lucky [name]
```

## Outdated

```sh
mas outdated
```

## Upgrade

```sh
mas upgrade
```
