# Services

## Installation

```sh
brew tap homebrew/services
```

## List

```sh
brew services list
```

## Run

```sh
brew services run [name]
```

## Start

```sh
brew services start [name]
```

## Stop

```sh
brew services stop [name]
```

## Restart

```sh
brew services restart [name]
```

## Cleanup

```sh
brew services cleanup
```
