# Tips

## From Repo

```sh
go mod init github.com/[username]/[repository]
```

## Migration

```sh
go mod init
go mod tidy
rm -fr vendor/
go build
rm -f Gopkg.lock Gopkg.toml
git commit -m 'chore(dep): migrated from dep to Go 1.11 modules'
```
